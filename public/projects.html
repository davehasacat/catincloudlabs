Projects

Selected work and reference architectures.

Enterprise Stock Market ELT

A production-grade pipeline on AWS with MWAA (Airflow), Snowflake, and dbt Cloud. Private-by-default networking (S3 Gateway + interface endpoints), SSE-KMS, Secrets Manager for credentials, and fine-grained IAM — all aligned to an enterprise posture.

  • Incremental models in dbt for efficient transforms
  • Curated S3 buckets: artifacts, raw, stage, curated, snowflake-stage
  • Observability via CloudWatch logs and Airflow task monitoring

MWAA Private Environment

Pattern for securely hosting Airflow with private web server access, VPC endpoints for ECR/Logs/KMS/Secrets, and a hardened artifacts bucket. This is our baseline for any data platform requiring strict network boundaries.

Future: Dashboards & Apps

Roadmap includes interactive dashboards (Plotly/Streamlit) on AWS App Runner/ECS, fronted by Cloudflare for global delivery and secure edges.
